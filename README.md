# 📖 About Project

Wedding Management System is a web-based information system designed to assist Wedding Organizer companies in managing operational activities efficiently. The system digitizes business processes including event management, crew assignment, vendor management, payment tracking, client portal, financial reports, and digital invitations.

This project was developed as part of an undergraduate thesis using the **Prototype Development Method**, allowing continuous collaboration between developers and stakeholders during system development.
